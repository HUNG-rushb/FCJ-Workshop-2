### Truy cập file thành công

1. Trở về giao diện bucket, chọn vào file **S3_logging_workshop.txt**.

2. Copy **Object URL**.

3. Paste URL vào trình duyệt vào xác nhận đã truy cập được vào file. Hãy giữ lại URL này cho phần tiếp theo.
